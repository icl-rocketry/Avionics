\end{DoxyNote}

\begin{DoxyReturn}{Returns}
For success \mbox{\hyperlink{class_fat_file_aa4a5b81161994cea07938702cdfce49f}{write()}} returns the number of bytes written, always {\itshape nbyte}. If an error occurs, \mbox{\hyperlink{class_fat_file_aa4a5b81161994cea07938702cdfce49f}{write()}} returns -\/1. Possible errors include \mbox{\hyperlink{class_fat_file_aa4a5b81161994cea07938702cdfce49f}{write()}} is called before a file has been opened, write is called for a read-\/only file, device is full, a corrupt file system or an I/O error.
